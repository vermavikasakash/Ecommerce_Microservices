## Available Scripts

To start the database application you can run:

### `npm start`

# On Successful run

in console we will get success massage.

# Project structure

Backend follows the MVC pattern.

-1) controller folder - it will contain all the functions related with the routes
-2) routes folder - it will contain the routes of application
-3) server.js - main application file to run the backend operation
